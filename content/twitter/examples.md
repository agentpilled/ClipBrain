# Examples

## Short Posts

```text
Your AI has tools.

It can write code, search files, and call APIs.

But it still has no idea what you read yesterday.

I am building ClipBrain so Claude Code, Codex, and other MCP agents can use your saved reading as local, cited memory.
```

```text
Most notes apps are designed for humans to browse later.

I want ClipBrain to be memory agents can use now:
- retrieve the right source
- return citations
- surface related saved notes
- stay local-first
```

```text
Tiny product bar for ClipBrain:

A stranger should clone the repo, run setup, clip one page, ask their AI a question, and feel the click.

Anything before that moment is just plumbing.
```

## Thread Shape

1. Name the problem in one sentence.
2. Show why current workflows fail.
3. Show what ClipBrain does differently.
4. Point to a concrete build detail or demo.
5. End with a specific question or invitation.

## Reply Style

Replies should add a useful angle, not just promote ClipBrain.

Good:

```text
This is exactly where memory gets interesting. Tools let agents act, but saved context tells them what matters to you.
```

Bad:

```text
Check out ClipBrain!
```
