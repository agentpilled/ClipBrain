# Reply Targets

Draft-only input for `bun run twitter:draft`.

Copy this file to:

```bash
content/twitter/reply-targets.local.md
```

Paste public tweet URLs or short notes you want the agent to help with. Keep one
target per bullet. Add an optional angle after `|`.

```text
- https://x.com/someone/status/123456789 | Bridge MCP tools to memory with source-grounded context.
- Thread about local-first AI | Emphasize that local-first should still feel magical.
- Post about Kindle highlights | Connect highlights to agent memory without sounding like a notes app.
```

Do not paste DMs, private customer content, private captures, emails, or
anything you would not want reflected in a draft.
